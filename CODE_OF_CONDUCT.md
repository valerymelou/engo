# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its
code of conduct. Its full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Scope

The covenant applies to all project spaces — the repository, issue and
pull-request discussions, GitHub Discussions, and any other space where
someone is representing the project.

## Reporting

Concerns about conduct in the project can be reported privately to the
maintainer at **me@valerymelou.com**. Reports will be handled
confidentially and in good faith. You do not need to know for certain
that a rule was broken to report something that made you uncomfortable.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
